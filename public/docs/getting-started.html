Getting Started

Navio is designed to be dropped into any project with zero build step required.

Installation via CDN

<script src="https://cdn.jsdelivr.net/gh/fahmirizalbudi/navio@main/bundle.js"></script>

Quick Start

1

Initialize Router

Create a new instance. Use hashMode: true for static sites.

const router = new Navio({ hashMode: true });
2

Define Routes

Chain your route definitions clearly and concisely.

router
  .on('/', () => renderHome())
  .on('/about', () => renderAbout())
  .start();
