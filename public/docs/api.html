API Reference

Complete documentation for the Navio class.

new Navio(options)

Initializes a new router instance. You can configure the root path and the routing mode (History API vs Hash).

Parameters

options.root string Optional

The base path for the router. Useful if your app is hosted in a subdirectory. Defaults to '/'.

options.hashMode boolean Optional

Whether to use hash-based routing (e.g., /#/about). Defaults to false (History API).

// Example: Hash mode for static file hosting
const router = new Navio({ 
  root: '/', 
  hashMode: true 
});

.on(path, callback)

Registers a route handler. The callback receives a context object containing parsed parameters and query strings.

Context Object

  • context.path The current matched path string.
  • context.params Object containing dynamic route parameters (e.g., :id).
  • context.query Object containing parsed URL query parameters. 
router
  .on('/', () => console.log('Home'))
  
  // Dynamic route with parameter
  .on('/user/:id', (context) => {
    console.log(`User ID: ${context.params.id}`);
  })

  // Route with query string (e.g., /search?q=navio)
  .on('/search', (context) => {
    console.log(`Search Term: ${context.query.q}`);
  });

.notFound(callback)

Sets a fallback handler that executes when no registered route matches the current URL. 

router.notFound((context) => {
  document.body.innerHTML = '<h1>404 Not Found</h1>';
  console.error(`Missing path: ${context.path}`);
});

.navigate(path)

Programmatically navigates to a new path. This updates the browser URL and triggers the router resolution logic immediately.

router.navigate('/dashboard');

.start()

Initializes the router. This method attaches event listeners to popstate (or hashchange) and global click events for data-navio links. It then resolves the current route.

Navio.fetchView(url)

Static Async

A helper method to fetch an external HTML file and extract its <body> content. Useful for loading page partials. 

router.on('/about', async () => {
  const html = await Navio.fetchView('views/about.html');
  document.getElementById('app').innerHTML = html;
});

Declarative Navigation

Navio automatically intercepts clicks on anchor tags with the data-navio attribute, preventing full page reloads.

<a href="/profile" data-navio>Go to Profile</a>
